TaskFlow- - BACKEND é um projeto de estudo focado no desenvolvimento de um backend para gerenciamento de tarefas diárias.
O objetivo é criar uma API REST que permita integração com um frontend, explorando boas práticas de backend, organização de projeto e persistência de dados.

Objetivos do projeto

Praticar desenvolvimento backend com Django

Criar uma API REST para operações CRUD

Implementar autenticação de usuários

Integrar backend e frontend

Realizar deploy do backend em ambiente real

Stack utilizada

Linguagem: Python

Framework: Django

Banco de dados: SQLite

ORM: ORM nativa do Django

IDE: Visual Studio Code

Status do projeto
Em desenvolvimento (projeto de estudo)

Funcionalidades atuais:

"Autenticação  (Login, cadastro, logout)"
"API baseada em json"
"Persistencia com SQlite"
